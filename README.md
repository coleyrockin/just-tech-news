# Just tech news

Just Tech News: Tech news website where users can post, upvote, and comment on links to news articles.

## Usage

- Bcrypt
- Connect-session-sequelize
- Dotenv
- Express
- Express-handlebars
- Express-session
- Mysql2
- Sequelize
- Deployed on Heroku / Plugin: JawsDB MySQL
